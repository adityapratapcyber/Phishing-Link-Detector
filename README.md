# Phishing Link Detector

This is a simple Python project that checks URLs for common phishing indicators.

I created this project to understand how suspicious links can be identified using basic security checks.

## Features

- Detects URLs containing '@'
- Detects shortened links such as bit.ly
- Checks whether HTTPS is being used
- Beginner-friendly implementation

## Example

Input:
http://example.com

Output:
Warning: Website is not using HTTPS
